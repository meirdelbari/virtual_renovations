# Git Push Commands for Virtual Renovations

# 1. Add the remote repository (if you haven't already)
git remote add origin https://github.com/meirdelbari/virtual_renovations.git

# 2. Stage all changes
git add .

# 3. Commit the changes
git commit -m "Ready for Vercel deployment"

# 4. Rename branch to main (standard practice)
git branch -M main

# 5. Push to GitHub
git push -u origin main
